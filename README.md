# py2cwl

## Usage

### Python to CWL JSON
	python py2cwlwriter.py > <file.cwl>

### Python to CWL YAML (messy)
	python py2cwlwriter.py | json2yaml > <file.yml>

### purifyCWL
    python purifyCWL.py -j <input.json> -o <output_filename>
